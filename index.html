论文已录用 Advanced Science IF 14.1

DDSurfer

GitHub

dMRI 皮层表面重建

基于 DTI 标量图实现白质表面与软脑膜表面重建流程，覆盖预处理、双流网络推理、模板空间预测与 native-space 回传。相关工作已被 Advanced Science 录用，中科院一区 Top 期刊，影响因子 14.1。

PyTorchdMRISurface Reconstruction

DDSurfer 面向 diffusion MRI 派生标量图进行皮层表面重建，降低对传统 T1-only 流程的依赖，形成面向扩散影像场景的表面重建路线。

03
NC 在投 智能体工作流

dMRI-Agent

GitHub

协议驱动 dMRI 智能体系统

面向扩散 MRI 工作流的协议驱动型智能体系统，采用 planner / executor 协作与协议化流程控制，支持任务规划、QC 追踪和结构化日志审计；相关论文正在投 Nature Communications。

AgentdMRIWorkflow Automation

该系统将 diffusion MRI 工作流拆解为受协议约束的阶段，支持 planner / executor 协作、QC 追踪、结构化日志和可审计自动化，用于可复现神经影像流程。

04
Nature Protocols 在投 3D Slicer 扩展

SlicerDDSurfer

GitHub

3D Slicer 扩展

开发 3D Slicer 扩展模块，统一封装 DDSurfer、DDParcel、DDSeg 与 Microstructure 等 diffusion MRI 流程，提供输入校验、任务编排与界面入口；相关 protocol 论文正在投 Nature Protocols。

3D SlicerPythondMRI Pipeline

SlicerDDSurfer 将多条 diffusion MRI 流程封装到统一的 3D Slicer 界面中，重点处理输入校验、任务编排和后续纤维追踪、连接组分析、可视化扩展。
