DDSurfer

dMRI cortical surface reconstruction

A DTI-based pipeline for white and pial surface reconstruction, covering preprocessing, dual-stream inference, template-space prediction, and native-space back-projection. The related work has been accepted by Advanced Science, a JCR Q1 top-tier journal with Impact Factor 14.1.

dMRI-Agent

Protocol-driven agent system for diffusion MRI

A protocol-driven agent system for diffusion MRI workflows, using a planner/executor architecture with structured protocol control, task planning, QC tracking, and auditable logs. The related manuscript is submitted to Nature Communications.

SlicerDDSurfer

3D Slicer extension

A 3D Slicer extension that unifies DDSurfer, DDParcel, DDSeg, and Microstructure diffusion MRI workflows under a consistent interface with input validation and workflow orchestration. The protocol manuscript is submitted to Nature Protocols.
